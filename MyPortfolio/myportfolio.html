Projects


Automatic Bell Ringing System

This Application is designed to automate the College/School bell ringing system.
It manages all alarm types like Prayer songs, National Anthem, and all announcements, class ending
timings made by the authorities by college/school. The system can adopt all types of audio formats.
The set audio's will be played at the selected Time. This project eliminates the need of manual ringing and
ensures accurate timing, leading to more efficient time management.


Functionalities are:


Time Keeping: The RTC modules provides the system with the current time.

Schedule Programming: can program bell schedules using push buttons or other input methods.

Time Comparison: The microcontroller compares the current time (from the RTC) with the programmmed
bell times.

Bell Activation: When the current time matches a scheduled bell time, the microcontroller
activates the relay to ring the bell.

User Interface: The LCD display and push buttons provide a user interface for setting
adjusting bell timings.

  • Accuracy
  • Efficiency
  • Customization
  • User-Friendly
